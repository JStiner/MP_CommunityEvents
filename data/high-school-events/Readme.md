# High School Events

Standard data folder structure for the event site.

## Files
- `event.json` — event metadata, disclaimers, tabs, and split-file references
- `locations.json` — map and venue/location data
- `schedule.json` — event schedule items
- `vendors.json` — vendor list, or an empty array when not used
- `flyer.json` — printable flyer data; currently a stub until a flyer is built

This folder follows the same structure as `data/covh/`.

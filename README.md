# Community Events Calendar Demo

Static GitHub Pages style project with a soft, mobile-friendly interface and separate event data files.

## Included
- `index.html` home page
- `fall-fest.html`
- `second-fridays.html`
- `christmas-on-vinegar-hill.html`
- shared CSS and JS
- separate JSON data file for each event

## Update pattern
Each year, update the JSON files in `/data` with:
- event title and summary
- dates / day labels
- locations
- schedule items
- vendor info
